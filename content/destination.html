destination

Description

The destination attribute categorizes the target of a hyperlink.

A personalization agent or user agent may add additional familiar user interface features to help users understand the link and follow the right one easily. User agents might add a familiar customized icon to the link, style the link in a customized way, or position the link on the page in a location that the user generally expects to find a link of the particular type. Some of these token values can apply to both an Action and a Destination. Content authors should choose the appropriate component in context.

Example

The following is an example using the destination attribute.

        <a href="home.html" data-destination="home">our main page</a>

See the destination sample user experience.

Note that this attribute can only take a single token value, and there is no default value.
